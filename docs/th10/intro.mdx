# 동방풍신록 (Th10)

**東方風神録 ～ Mountain of Faith**

| | |
|---|---|
| 출시일 | 2007년 8월 17일 |
| 개발 | 상하이 앨리스 환악단 (ZUN) |
| 장르 | 탄막 슈팅 |

## 줄거리

:::caution 준비 중
게임 줄거리는 추후 업데이트 예정입니다.
:::

## 특징

- 아이템을 수집해 파워를 올리는 신 시스템 (구 파워 시스템 폐지)
- 신앙 게이지 — 그레이즈·적 처치로 채워지는 점수 배율
- 레이무·마리사 두 캐릭터, 각각 세 가지 샷 타입
