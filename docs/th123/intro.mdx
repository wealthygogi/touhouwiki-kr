# 동방비상천칙 (Th123)

**東方非想天則 ～ 超弩級ギニョルの謎を追え**

| | |
|---|---|
| 출시일 | 2009년 8월 15일 |
| 개발 | 상하이 앨리스 환악단 (ZUN) |
| 장르 | 대전 격투 |

## 줄거리

그날 아침은 햇빛이 가려져 약간 어두웠다.
이런 여름에 계절에 맞지 않는 아침 안개가 낀 것일까.

더위를 잊게 해줄 안개를 기대하며 집을 뛰쳐나온 그대는,
눈앞에서 흔들리는 "그것"을 보고 분명 경악했을 것이다.

사람 형태를 한 "그것"은 미지근하고 불쾌한 증기를 내뿜으며,
산을 올려다보듯 흔들흔들 움직이고 있었다.

――늘 그랬듯 덥고, 늘 그랬듯 시끄러운 여름의 산.
그녀는 사람 형태의 "그것"을 내려다보고 있었다.
