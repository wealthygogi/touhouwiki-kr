|---|---|

| 개발 | 상하이 앨리스 환악단 (ZUN) |

## 특징

- 인간+요괴 4팀(8캐릭터) 중 선택
- 시간 제한 시스템 — 인간 캐릭터로 보스 처치 시 추가 시간 획득
- 히든 스테이지 '단야 루트' 포함 8개 엔딩
