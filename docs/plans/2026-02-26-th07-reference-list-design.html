Design Document: TH07 Reference List for MDX Conversion

Date: 2026-02-26
Status: Approved (via system directive)
Topic: TH07 Character and BGM Reference List

1. Purpose​

To provide a standardized reference for Korean character names and BGM titles for Touhou 07: Perfect Cherry Blossom (동방요요몽). This ensures consistency and accuracy when converting legacy text files or raw transcriptions into MDX/Docusaurus format, specifically for <Dialogue speaker='...'> and <Action>♪ BGM</Action> components.

2. Context​

The project involves migrating Touhou project data (dialogues, omake, music comments) to a Docusaurus-based wiki. Accuracy in speaker names and BGM titles is critical for the Korean localization.

4. Implementation Guidelines​

  • Dialogue Tags: Use the exact Korean name from the table (e.g., <Dialogue speaker='하쿠레이 레이무'>).
  • BGM Lines: Use the format <Action>♪ [BGM Title]</Action> (e.g., <Action>♪ 네크로 판타지아</Action>).
  • Standardization: "무하유지향" is preferred over "이상향" for Stage 1 theme to align with the normalized codebase data.
