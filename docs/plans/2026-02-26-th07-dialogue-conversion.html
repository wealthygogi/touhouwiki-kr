th07 Dialogue MD to MDX Conversion Implementation Plan

For Claude: REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

Goal: Convert 10 dialogue files in docs/mainline/gen1/th07/dialogue/ from .md to .mdx with high precision, moving comments to props, standardizing BGM titles, and adding correct anchors and sidebar positions.

Architecture:

  • Use <Dialogue speaker="..."> and <Action> components.
  • Move translator notes/comments (after ......, in parentheses, or at the end of a line) to the comment prop.
  • Standardize BGM titles: Keep the title and ♪ symbol, but remove external markdown links. e.g. <Action>♪ 크리스탈라이즈 실버</Action>
  • Use double quotes for attributes. Single quotes inside comments.
  • Set sidebar_position according to specific rules.
  • Update scenario ending links with anchors (e.g., #굿-엔딩-no-4-레이무-a).
  • Precision Example: | 마리사 | 오 빠 꺄마라드...... (Au Pas Camarades) ... | -> <Dialogue speaker="마리사" comment="(Au Pas Camarades) 리듬을 맞춰라 라는 의미, 프랑스 동요 '클라리넷을 고장냈어' 가사 중 일부">오 빠 꺄마라드......</Dialogue>

Tech Stack: MDX, Docusaurus components.

Task 3: Convert Endings & Overview (Wave 3)​

Files:

  • Modify/Rename: docs/mainline/gen1/th07/dialogue/reimu-endings.md -> reimu-endings.mdx
  • Modify/Rename: docs/mainline/gen1/th07/dialogue/marisa-endings.md -> marisa-endings.mdx
  • Modify/Rename: docs/mainline/gen1/th07/dialogue/sakuya-endings.md -> sakuya-endings.mdx
  • Modify/Rename: docs/mainline/gen1/th07/dialogue/overview.md -> overview.mdx

Rules:

  • reimu-endings.mdx: sidebar_position: 3
  • marisa-endings.mdx: sidebar_position: 6
  • sakuya-endings.mdx: sidebar_position: 9
  • overview.mdx: sidebar_position: 1
  • Endings files use <tl$Speaker:> tags in the second column. These must be parsed into <Dialogue speaker="Speaker">.
  • <l$> should be removed (usually means newline in raw script but MDX handles it via component/text).
  • <c$TEXT$> should be handled or stripped if redundant. (e.g., <c$ENDING No. 4$> -> can be ignored if the heading is already there, or moved to an Action).
