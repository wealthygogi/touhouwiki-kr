| 화자 | 대사 |
| --- | --- |
|  | 여름의 하쿠레이 신사.@  　동쪽 경계에 있는 신사...... 였지만, 지금은 완전히 박살나 있다.@ 　신사로서의 기능은 잃지 않았지만 주거지로서의 기능은 전무했다. |
|  | 신사가 무너진 원인은 직하형 대지진이었다고 한다.@  　사쿠야가 천계에 가 있을 동안 일어났던 건지, 　지진을 전혀 느낄 수가 없었다. |
|  | 레이무 "아~젠장, 나보고 어쩌라는 거야!"@  마리사 "멋지게 지붕이 내려앉았구만."@ 사쿠야 "지진으로 건물이 무너지면 지붕이 내려앉는 일도 흔히 있는 거네." |
|  | 마리사 "신사 지붕은 꽤 무거워 보였으니까 말이지."@  레이무 "다들 너무 냉정해!"@ 사쿠야 "홍마관에서는 진동을 전혀 느끼지 못했다고 그러길래." |
|  | 레이무 "아으, 이걸 어쩌면 좋냐......"@  마리사 "기부금이라도 모아보는 거 어때? 　　　　새전함이라도 매고 다니면서." |
|  | 사쿠야 "아, 나한테 좋은 생각이 있어." |
|  | 사쿠야 "실은 최근에 지진을 다루는 사람을 알게 돼서,"@  마리사 "......무슨 자다가 봉창 두들기는 소리야."@ 사쿠야 "그 사람한테 부탁하면 여진 걱정은 안 해도 될 걸." |
|  | 레이무 "이렇게까지 무너졌는데 여진 따위가 무섭겠냐!@ 　　　　지진이 오기 전에 소개해줬음 좀 좋아." |
|  | 사쿠야 "그런데 그게 말이야,@  　　　　내 생각인데 혹시 지진을 　　　　그 쪽이 일으킨 게 아닐까 싶어서 말이지." |
|  | 레이무 "!? 자세히 설명해봐!"@  사쿠야 "우선 신사가 무너질 정도의 지진이었잖아?@ 　　　　그런데 아무도 눈치채질 못했다는 게 좀 이상해." |
|  | 사쿠야 "그 사람은 국지적으로 지진을 일으킬 줄도 알더라.@  　　　　거기다 평소에도 '아~지진내고 싶다~'가 　　　　입버릇이었고." |
|  | 마리사 "뭐~? 그거 진짜냐?"@  사쿠야 "진짜일지 어떨지는 네가 결정하면 돼."@ 레이무 "......그 자식이 범인이야! 그 놈 어디 있어!?" |
|  | 사쿠야 "구름 위에 살고 있어"@  레이무 "구름 위......어디있든 가서 박살내주겠어! 　　　　그 놈 잡아다가 신사 물어내게 해야지!" |
|  | 사쿠야는 저번 싸움에서 정신나간 천인을 한바탕 혼을 내 줬지만...... 　아직도 분이 덜 풀린 참이었다. |
|  | 그래서 레이무를 보내서 한층 더 혼쭐을 내줘야겠다고 생각했다.  　레이무라면 자기보다 더 물불 안 가리고 　묵사발을 낼 게 분명했기 때문이었다. |
|  | 인간에게 질 리 없다고 큰소리 떵떵 치던 천인이 　레이무 손에 박살날 걸 생각하니 |
|  | 조금은 속이 풀린 듯한 느낌이 들었다. |
