|  | 얼마 지나지 않아 부하에게 연락이 왔다.\.  인간이 환상향에 들어올 준비를 시작했다.\. 환상향 측도 준비에 만전을 기했다. |

|  | 후자는 제한이 없다.\.  바깥세계의 인간은 전자의 방법으로 간섭하고 있어서\. 손을 댈 수가 없었던 것이다. |
