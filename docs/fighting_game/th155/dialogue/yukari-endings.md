## 엔딩 1

| 화자 | 대사 |
| --- | --- |
|  | 하쿠레이 신사.  환상향의 동쪽에 있고, 인간 마을에서 떨어진 신사이다. 늘 그렇듯 참배객은 보이지 않는다. |
|  | 마리사: 왜 그래?    왠지 힘이 없구만. 레이무: 진짜, 참배객도 안 오고   전혀 벌이가 안 돼. |
|  | 마리사: 하루이틀 그런 것도 아니잖아? 레이무: 이변해결을 해봐도 돈벌이는 안 되고...... |
|  | 마리사: 그러고 보니 완전빙의이변은 해결한 거야? 레이무: 응, 뭐 어떻게든. |
|  | 마리사: 대단한데.   어떻게 그 최흉최악의 두 사람을 쓰러뜨린 거야? |
|  | 레이무는 경위를 설명했다.  꿈의 세계에 간섭하여 가난신을 마스터로 삼음으로써 가난신의 강제빙의를 막았다고 말해주었다. |
|  | 마리사: 마스터와 슬레이브의 경계를 가지고 놀았다고?    그런게 가능한 녀석은 유카리밖에 없구만.   그래서 태그를 짠 건가. |
|  | 레이무: 분하지만, 유카리가 없었다면    분명 이길 수 없었겠지. 마리사: 뭐, 결과적으로 이겼으니 좋은 거 아니냐.   그래서. 중요한 건 그 두 명은 지금 어쩌고 있어? |
|  | 레이무: 그게 말이지...... |
|  | 묘렌사.  뱌쿠렌 무리가 세운 절. 인간, 요괴를 불문하고 존경받고 있다. |
|  | 히지리: 봐봐, 아직 더럽잖아요. 조온: 왜 내가 청소따윌 해야 하는 건데~ |
|  | 히지리: 청소따위도 못해서 어디다 쓰겠어요.    일상생활 하나하나가 다 수행이랍니다. 조온: 젠장~   인색하게 굴지 말라고 |
|  | 조온: 이런 검소한 절 따위에 있으면    부를 빼앗는 능력이 전혀 쓸모 없잖아. 히지리: 정원청소를 끝내면, 물을 길러오고, 명상하고   식사를 준비해야 합니다. 농땡이부릴 여유가 없어요. |
|  | 조온: 빨리 끝내고 도망쳐야지.  히지리: 도망칠 수 있다고 생각하면 큰 착각입니다.   착실하게 검소한 생활의 소중함을   철저히 주입시켜 주겠어요! |
|  | ............ |
|  | 마리사: 그렇군.    화려하게 낭비하던 녀석은 절에서 근성을 고치는 건가.   그건 효과가 있을지도. |
|  | 레이무: 원래는 감금이나 추방이었겠지만 말야.    하지만 역병신이라면 갱생할 수 있다고 뱌쿠렌이 말했거든.   바라던 바는 아니지만 맡기기로 했어. |
|  | 마리사: 그렇다면 나머지 한 명,   가난신은 어떻게 된 거야? |
|  | 레이무: 가난신은 말이지......   스스로 부를 모아 낭비하는 역병신보다도 성가시단 말야. |
|  | 마리사: 재물운을 불운으로 바꿔버리니    절망적으로 구제불능인 능력이니깐. 레이무: 그뿐만이 아니라, 아무래도   불운을 몸 속에 계속 쌓아두고 있는 모양이라서 |
|  | 레이무: 때때로 폭발적인 불운을 발산한대.    전투중에서도 그 발작이 일어나서 말야,   그 기운의 양은 진짜 장난이 아니었지. 마리사: 으앗. 가까이 하기도 싫구만. |
|  | 레이무: 그럼에도 아무도 떠맡을 사람이 없어서 말야.    ......그래서 어쩔 수 없었어. 마리사: 그런가, 뭐 그것도 어쩔 수 없지......   재난이니깐. |
|  | 시온: 청소 끝났어요~    다음은 뭘 하면 좋을까요? 레이무: 벌써 끝났어?   그럼, 이제 물 좀 길어와줘. |
|  | 마리사: 엥?! 이녀석, 왜 여기에 있는 거야?  레이무: 아무도 떠맡지 않아서 우리집에서 감시하게 됐어.   일은 제대로 하기도 하고, 꽤 도움이 돼. |
|  | 시온: 신사는 안락해서 좋네요오.    빈곤해져도 즐겁게 살아갈 수 있고. 레이무: 애초에 잃을 재산따윈 없으니깐.   것보다 시끄럽거든! |
|  | 마리사: 뭐야, 완전히 성불하거나 추방시킨 줄 알았어. 레이무: 간단히 성불시켜버리면 아무 고민도 없지만 말이지. |
|  | 레이무: 결국, 내가 가난뽑기를 뽑은 처지가 되어버렸네.  시온: 가난신이니까 말이죠.   것보다 시끄럽거든요. |
|  | 라이브회장에서 관객들로부터 부를 등쳐먹던 역병신과 가난신이라는 최흉최악의 두 사람. |
|  | 유카리와 레이무의 작전으로 완패한 결과,  역병신은 절에서 가난신은 신사에서 갱생시키게 되었다. 최흉최악의 두 사람의 위협은 사라졌다. |
|  | 마리사: 하지만 말야,    흑막을 회유했기도 했고,   완전빙의이변은 이걸로 해결이구만! |
|  | 레이무: 그러면 좋겠는데 말야. 마리사: 아직 뭔가 남았어? |
|  | 레이무: 역병신, 가난신을 감시하에 놓은 건 이유가 있어서야.   이변의 뒷처리를 해줘야 하거든. |
