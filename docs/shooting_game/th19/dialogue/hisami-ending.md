## 엔딩

|  |  |
| --- | --- |
|  | 메시지 #1 |
|  | 지옥。 |
|  | 다른 어느 곳보다도 넓은 공간이다。 |
|  | 그러나, 지상의 영토 다툼은 여기에 미치지 못한다。 |
|  |  |
|  | 잔무가 있는 곳에는 몇몇 요괴들이 불려져 있었다。 |
|  | 뭔가 즐거워 보이는 듯이。 |
|  |  |
| 히사미 | 「잔무 님이랑 저 허접들이 무슨 이야기를 하고 있는걸까나-。 |
|  | 왜, 나는 저기에 끼어들지 못한걸까。 |
|  | 이렇게 노력 중인데-」 |
|  |  |
| 히사미 | 「아, 술이다! |
|  | 저 녀석들, 잔무 님이랑 술 잔치를……。 |
|  | 설마 승리의 연회!? |
|  | 키-, 분해라-」 |
|  |  |
| 히사미 | 「근데, 뭐에 대한 승리지? |
|  | 저런 허접 자식들로 뭘 이뤘다는 거야? |
|  | 키-」 |
|  |  |
| 히사미 | 「……어? 잔무 님!?」 |
|  |  |
|  | 히사미는, 멀리서부터 잔무에게 불려져 동요했다。 |
|  | 이변 해결의 연회에 참가해도 좋다는 것이었다。 |
|  | 그녀는 기뻐했다。 |
|  | 그리고 동시에 놀라기도 했다。 |
|  |  |
|  | 이변이 해결 됐다고? |
|  | 애초에 이변은 뭐였지? |
|  | 잔무 님은 손바닥 위에서 뭘 하고 계셨던 거지? |
|  |  |
|  | 뭐, 그런 거 알아봤자 어찌 할 수도 없고。 |
|  | 난 그저 그런 지옥의 안내인이니까 말이지。 |
|  | 지상에서 활약하는 일은 절대로 없을테니까。 |
|  | 잔무 님의 활약만 볼 수 있다면 그걸로 만족이니까。 |
