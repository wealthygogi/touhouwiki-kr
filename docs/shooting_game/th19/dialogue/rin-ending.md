|  | 메시지 #1 |
|  | 옛 피연못지옥. |
|  | 옛 작열지옥의 바닥에서, 옛 지옥 중에서도 가장 꺼릴 수밖에 없는 장소가 있다. |
|  | 좀처럼 탈탄소가 진행되지 않는 것은 여기에 저주가 걸려 있기 때문일까. |
|  |  |
| 치야리 | "……그래 토테츠를 만났구나" |
| 린 | "그래, 그 일만 있었던 건 아니지만……. |
|  | 애초에, 너는 왜 강욕동맹에 들어간 거야?" |
|  |  |
| 치야리 | "으음, 재밌어 보였으니까? 라던가?" |
| 린 | "지-이" |
| 치야리 | "그 느낌, 뭔가 말하고 싶지 않은데에" |
|  |  |
| 린 | "토테츠는 너를 신뢰하지 않는 것 같았어. |
|  | 그 이유는, 뭔가 감추고 있고 뭔가 함정이 장치되어 있어서라던가……" |
| 치야리 | "뜨끔, 설마 들켰던 거야? |
|  | 토테츠가 무슨 말 했어?" |
|  |  |
| 린 | "처음부터 널 신뢰하지 않은 듯 했었고, 내 알 바 아니라던 느낌이었지. |
|  | 그건 그렇고 너, 무슨 음모를 꾸미고 있는 거야? |
|  |  |
| 치야리 | "덫은 말이야, 내가 둔 게 아니야. 거래 조건의 하나였다구" |
| 린 | "거래? 혹시, 잔무하고?" |
|  |  |
|  | 옛 지옥에서 닛파쿠 잔무의 이름을 모르는 자는 없다. |
|  | 낡은 지옥을 개혁하고 옛 지옥을 버리고 신천지를 노리자는 아이디어를 낸 것은, 다른 그 누구도 아닌 잔무였으니까. |
|  |  |
|  | 많은 오니와 요괴들이 찬성했지만, 기득권의 권익에 매달린 일부 오니와 요괴는 그것에 반항했다. |
|  | 즉, 지금 옛 지옥에 남아있는 자들은 반 잔무 세력이 많은 것이다. |
|  |  |
|  | 지상에서 일어나고 있는 대지의 이변은, 바로 그것과 똑같은 것이라 생각되었다. |
|  | 잔무가 뒤에서 움직이고 있는 것이라고 한다면 전부 납득할 수 있다. |
|  | 무슨 일이 일어나고 있는지는 잘 판단되지 않았지만……。 |
