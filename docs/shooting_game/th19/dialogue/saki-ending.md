|  | 꺄- 멋있어-! 역시 오니는 이래야지!」 |

|  | 에노코에겐 잔무의 입김이 닿았다는 거네. |
|  | 더 감시할 필요가 있다...는 건가」 |
|  |  |
|  | 사키는 진작에 전의를 상실했다. |
|  | 잔무라고 하는 오니는, 지옥의 인기인이면서 축생계에서도 이름이 알려져있는 카리스마의 상징이었다. |
|  |  |
|  | 에노코에겐 잔무의 입김이 서렸다고 생각했다. |
|  | 사키는 늑대령들을 사용해 감시 하도록 했으나 당장 이 이상의 것들은 알지 못했다. |
|  |  |
|  | 하지만, 에노코의 출생엔 자신도 알지 못했던 아주 큰 비밀이 하나 있었다. |
|  | 물론, 사키는 알지도 못하겠지만. |
|  | 그걸 알고 있는 건 잔무 뿐이었다. |
