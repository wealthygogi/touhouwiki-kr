## 엔딩

|  |  |
| --- | --- |
|  | 메시지 #1 |
|  | 하쿠레이 신사. |
|  | 동쪽 변두리에 있는 신사. |
|  | 교통이 불편한데도, 대부분의 이변 관계자들이 모이는 곳이기도 하다. |
|  |  |
| 레이무 | "……이변 조사의 진척? |
|  | 그래, 뭐 그, 순조로운데. 왜 그런 걸 묻는 거야?" |
| 츠카사 | "꽤나 광범위하고 또 많은 이들이 연관된 이변인 듯해서, |
|  | 혹시 곤란을 겪고 있다면 협력해 드릴까 해서요" |
|  |  |
| 레이무 | "협력이라니…… 무슨 바람이 분 걸까" |
| 츠카사 | "왜냐하면, 제가 조사한 바로는" |
|  | 킷초가 이끄는 귀걸조, 쿠로코마가 이끄는 경아조, 토테츠가 이끄는 강욕동맹에 |
|  | 다양한 세력이 얽혀 있어요." |
|  |  |
| 츠카사 | "그리고…… 그 이외의 미지의 세력도. |
|  | 아무리 레이무 씨라도 힘에 부치지 않을까 해서" |
| 레이무 | "흥, 누가 부추긴 거야?" |
| 츠카사 | "아니, 부추김이라뇨" |
|  |  |
| 레이무 | "맞혀볼까. 너, 지옥에서 오니랑 만났지" |
| 츠카사 | "무슨" |
|  |  |
| 레이무 | "그녀석이 말하기를, 이미 이변은 해결되고 있는 것 같아. |
|  | 나한테는 아직 보이지 않지만, 그렇구나 싶었지. |
|  | 순조롭게 진행되면, 내 쪽에 자객이 온다고도 했었지. |
|  | 아무래도 널 말하는 거겠지" |
|  |  |
|  | 츠카사는 한 마디도 말 못하고 입을 다물어 버렸다. |
|  | 레이무는 그것을 보고 같은 오니를 만났던 때를 떠올렸으리라. |
|  | 사람을 속이는 자에게 침묵이란 치명적이다. |
|  | 더는 레이무를 속이는 것은 불가능하다. |
|  |  |
|  | 레이무를 보니 이미 오니하고는 승부가 난 듯 보였다. |
|  | 그 사실은, 레이무 토벌 의뢰는 진심이 아니라는 것을 의미하고 있겠지. |
|  | 그 오니의 장난에 농락당한 듯하다. |
|  |  |
|  | 덜컥 피곤함이 몰려온 츠카사였지만, 이 이변을 통해 한 밑천 잡을 수 있었던 것은 틀림이 없다. |
|  | 전쟁은 늘 그렇듯 요령 좋은 자들의 장사 수단인 것이다. |
