## 치르노 (チルノ, Cirno)

<img src={cirno} alt="치르노" style={{width: 250}} />

　○녹지 않는 무더운 날의 얼음 요정  
　　치르노  
　　Cirno  

　　종족: 요정  
　　능력: 얼음을 조종하는 정도의 능력  

　　호수 근처에서 떼지어다니는 얼음 요정.  

　　어째서인지 한여름의 더위에도 당하지 않는 힘을 손에 넣어서 정신적으로 무적상태가 되었다.  
　　딱히 이변이 일어났다고 생각하지는 않지만 마음이 들뜬 사이에  
　　세상을 여행하고 싶어졌다. 단지 그것 뿐이었다.
