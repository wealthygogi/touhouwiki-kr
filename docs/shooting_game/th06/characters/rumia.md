　　사실 머리카락에 감고 있는 리본은 부적이고, 리본은 떼고 싶어도  
　　루미아 본인은 만지는 것도 불가능합니다.  

　　실제로 어둠을 다루는 장면은 볼 수 없지만  
　　사실 루미아 주변은 항상 밤입니다.
