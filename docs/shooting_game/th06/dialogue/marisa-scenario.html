레밀리아 스칼렛 등장
영원히 붉은 어린 달 레밀리아 스칼렛
레밀리아재주 있는 매는 꼬리를 숨기지 않는다는 거지. 재주 있는 매는 발톱을 숨긴다(能ある鷹は爪を隠す)와 꿩은 머리는 숨기고 꼬리를 숨기지 않는다(頭隠して尻隠さず)를 섞은 것
마리사매가 문제가 아니라 넌 뇌가 없는 것 같은데. 재주(能)와 뇌(腦)의 발음이 같음. 의역하였다.
레밀리아뇌같이 단순하고 과학적인 사고 중추가 필요한 건
레밀리아인간 뿐이야.
