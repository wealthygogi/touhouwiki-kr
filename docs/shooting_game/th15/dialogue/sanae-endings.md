## 완전무결/레거시 노미스 엔딩 No. 05
|  |  |
| --- | --- |
|  | 모리야 신사. |
|  | 요괴의 산에 있는 비교적 새 신사이다. |
| 사나에 | "'── 그 순간, 나의 최종오의가 거대한 악을 무찌르고, |
|  | 달의 도시는 구원받게 된 것이다. |
|  | 이리 하여, 나는 달의 구세주로서 추앙받게 되었다.'" |
| 사나에 | "대충 이 정도면 되려나? |
|  | 그리고 알아보기 쉽게 일러스트 같은 것도 넣어야지." |
| 카나코 | "...... 그래서, 이게 이번 사건의 보고서?" |
| 사나에 | "네, 제 무용담이에요." |
| 카나코 | "흐음...... 내용이 꽤 경악스러운데, 어디까지가 사실인 거지?" |
| 사나에 | "무려 80% 사실입니다." |
| 카나코 | "...... 이제부터 보고서는 100% 사실로만 적도록." |
| 카나코 | "...... 흠, 흠. |
|  | 달의 도시가 환상향으로 천도하려고 했다...... |
|  | 다만 그것은 달의 도시가 침략당한 까닭이었다, 이게 정말이야?" |
| 사나에 | "네, 그건 사실이에요." |
| 카나코 | "이게 사실이라면, |
|  | 달의 주민들이 너에게 사건을 의뢰한 이유는......" |
| 사나에 | "아마 제가 강해서였겠지요." |
| 카나코 | "자기 힘으로는 어찌할 수 없는 적이어서, 라는 말이 되는군." |
| 카나코 | "...... 이 사구메라는 자, 혹시......" |
| 사나에 | "아시는 이름인가요? 좀 이상한 녀석이었는데." |
| 카나코 | "으음... 사구메라면, 달의 주민들은 천신의......" |
|  | 거기까지 말하고, 카나코는 혼자 생각에 잠겼다. |
|  | 아무래도, 달의 주민들은 신의 계보를 잇는 자들이 아닌가 하는 것이다. |
|  | 그 자들이 마지못해 그렇다고는 하지만, 어쨌든 지상으로 내려오려 했다. |
|  | 그렇다면 이것은 천손강림의 재래가 아닌가. |
|  | 자각하지는 못 했지만, 사나에가 그것을 저지했다는 건 토착신으로서 칭찬받아 마땅할 것이다. |
|  | '역시 이 아이에겐 뭔가 있다' 라고 카나코는 생각했다. |

## 레거시 엔딩 No. 06
|  |  |
| --- | --- |
|  | 모리야 신사. |
|  | 요괴의 산에 있는 비교적 새 신사이다. |
| 카나코 | "이런 것도 못 피하다니! |
|  | 호수 한 바퀴 돌고 와!" |
| 사나에 | "네! 다녀오겠습니다!" |
|  | ............ 수십 분 후. |
| 카나코 | "느려 터졌어! |
|  | 다음 탄막 세트 간다!" |
| 사나에 | "하아, 하아... 조금만 쉬었다 하면 안 될까요?" |
| 카나코 | "그런 근성 갖고 달의 주민과 대적할 수 있다고 생각하느냐! |
|  | 또 엉망진창으로 당해서 돌아오고 싶은 거냐!?" |
| 사나에 | "아뇨오... 그건 싫지만." |
| 사나에 | "스포츠에 근성론이라니, 이제 그런 시대도 아니고......" |
| 카나코 | "아직도 모르는구만! |
|  | 그런 시대가 아니니까 중요한 것이다!" |
| 카나코 | "따져 보면 탄막은 근성! |
|  | 피하지 못하는 것은 근성이 부족한 탓! 알았느냐!?" |
| 사나에 | "네에......" |
| 스와코 | "분석 결과 나왔어~" |
| 카나코 | "어때?" |
| 사나에 | "하아... 하아, 헉..... 분석이라뇨?" |
| 스와코 | "뭐긴, 당연히 '감주의 약'얘기지. |
|  | 사나에가 달에 가 있는 동안 이 약을 조사해 봤거든." |
| 사나에 | "하아... 어찌 되었건 전 약을 쓰지 않고 갈 수밖에 없던 거네요." |
| 스와코 | "그래도, 쓰지 않길 잘한 것 같아. |
|  | 이 약에는 강한 부작용이 있을 가능성이 높아." |
| 카나코 | "호오...... 어떤 부작용?" |
| 스와코 | "미래 체험을 가능하게 함과 동시에, 과거를 정화하는 효과도 확인됐어. |
|  | 오랜 토착신령들에게 시험해 봤더니......" |
| 카나코 | "한 방에 나가떨어졌다 이건가." |
| 스와코 | "맞아, 이 약은 지상에 사는 생물에게는 영향이 세." |
| 사나에 | "네? 그럼 혹시 제가 썼더라면 위험할 뻔했던 건가요?" |
| 스와코 | "한 방...... 까지는 아니겠지만." |
| 스와코 | "그렇다 하더라도 분명 필요 이상의 불결함이 정화돼서 깨끗한 인간이 되었을지도 모르지." |
| 사나에 | "......? 그건 좋은 거 아닌가요?" |
|  | 지상인과 달의 주민의 차이는 더러움인가 아닌가의 차이이다. |
|  | 더럽혀진 지상을 버리고 달로 옮겨간 자가 달의 주민이기 때문이다. |
|  | 달의 주민이 말하는 불결함이란, 태어나는 것과 죽는 것. |
|  | 그 불결함을 잃는다는 것은, 삶과 죽음의 양측을 잃는다는 것이다. |
|  | 그렇게 되면 생명이 넘쳐 흐르는 지상에서는 살 수 없게 되리라. |
|  | 감주의 약은 생과 사의 존재를 희박하게 한다. 분명 그랬다. |
|  | 잔기라고 하는 개념을 없애고, 미스가 가벼워짐을 느꼈을 것이다. |
|  | 감주의 약을 계속 사용하는 것은 위험할지도 모른다. |
