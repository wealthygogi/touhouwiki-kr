# 동방빙의화 (Th155)

**東方憑依華 ～ Antinomy of Common Flowers**

| | |
|---|---|
| 출시일 | 2017년 12월 29일 |
| 개발 | 상하이 앨리스 환악단 (ZUN) |
| 장르 | 대전 격투 |

## 제작자 코멘트

안녕하세요, ZUN입니다.
완전빙의라는 색다른 태그매치를 즐겨주시길 바랍니다.

도시전설이변의 일부라고 생각되는 완전빙의,
이변이라고 불릴 만한 일이 벌어졌습니다.
이번에는 플레이를 시작하면 왠지 답답한 사건들이 계속 쌓여가고,
마지막에는 폭발하는 듯한 스토리를 지향했습니다.
이변 자체는 매우 간단하지만 해결될 때까지 계속해서 문제가 일어나죠.
그리고, 어째서 마지막은 이런 식으로 되는 거야? 라고 웃어주셨으면 좋겠습니다.

성능만 보고 콤비를 선택하는 것도 좋지만, 개성적인 캐릭터도 갖춰져 있으니,
부디 자기 나름대로 재미있을 것 같은 조합으로 즐겨보는 건 어떨지?

ZUN（아들과 딸, 심지어 유아도 관람할 수 있는 클래식 콘서트에 갔다 왔습니다)
