# 치르노의 ⑨엔 모금 대작전

하쿠레이 신사의 기부함이 텅 비어있다! 최강의 천재 치르노가 ⑨엔씩 모금 운동을 시작합니다.

:::info 규칙
- **10초** 안에 최대한 많이 클릭(터치)하세요!
- 클릭할 때마다 **100엔**씩 모금됩니다
- 모금액에 **숫자 9**가 포함되면 (900엔, 1900엔, 9000엔...) 동방 시리즈 이펙트가 발동합니다
- 최고 기록은 브라우저에 저장됩니다. 다시 도전해보세요!
:::
