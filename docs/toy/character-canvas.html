캐릭터 캔버스

동방 프로젝트의 다양한 캐릭터들을 선택하고 캔버스에 자유롭게 배치하여 나만의 이미지를 만들어보세요!

사용 방법​

  1. 게임 선택: 상단의 첫 번째 셀렉트박스에서 게임을 선택합니다
  2. 캐릭터 선택: 두 번째 셀렉트박스에서 원하는 캐릭터를 선택합니다
  3. 캐릭터 배치: 선택한 캐릭터가 캔버스에 나타납니다
  4. 조작: 캐릭터를 드래그하여 이동하고, 크기를 조절할 수 있습니다
  5. 삭제: 우측 상단의 - 버튼을 클릭하거나 Delete 키를 눌러 캐릭터를 제거할 수 있습니다
  6. 줌: 마우스 휠을 사용하여 캔버스를 확대/축소할 수 있습니다

주요 기능​

📝 텍스트 추가​

  • 텍스트 입력창에 원하는 문자열 입력
  • 컬러 피커로 텍스트 색상 선택
  • Enter 키 또는 "텍스트 추가" 버튼으로 추가
  • 텍스트도 자유롭게 이동, 리사이징, 삭제 가능

📥 PNG 다운로드​

  • 캔버스 내용을 고해상도 PNG 이미지로 저장
  • 격자 배경 없는 깔끔한 이미지 생성

⌨️ 키보드 단축키​

  • Delete/Backspace: 선택된 요소 삭제

팁​

  • 여러 캐릭터를 배치하여 다양한 구도를 만들어보세요
  • 텍스트를 추가하여 제목이나 주석을 넣어보세요
  • 캐릭터의 크기를 조절하여 원근감을 표현할 수 있습니다
  • 마우스 휠을 사용하여 세밀한 작업을 할 수 있습니다
  • Delete 키를 사용하여 빠르게 요소를 삭제할 수 있습니다
