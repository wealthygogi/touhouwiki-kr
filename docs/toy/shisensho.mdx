# 사천성 (四川省)

동방 프로젝트 캐릭터 16x16 도트를 타일로 사용한 사천성 게임입니다.

## 규칙

1. 같은 캐릭터 타일 2개를 선택하면 제거됩니다
2. 두 타일은 **최대 2번 꺾이는 경로**로 연결 가능해야 합니다
3. 경로는 다른 타일을 통과할 수 없습니다
4. 모든 타일을 제거하면 승리!

:::tip 팁
게임 진행 중 매칭 가능한 쌍이 없어질 수 있습니다. 이 경우 **섞기** 버튼을 사용하면 남은 타일이 재배치됩니다. 힌트와 섞기는 각각 3회씩 사용할 수 있으니 아껴서 쓰세요!

**어려움**과 **루나틱** 난이도는 타일이 많아 모바일에서는 **가로 모드**를 권장합니다.
:::
