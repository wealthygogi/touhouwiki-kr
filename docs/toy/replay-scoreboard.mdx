## 동방프로젝트 리플레이 스코어보드

- 동방프로젝트 리플레이 스코어보드
- 東方Project リプレイスコアボード
- Touhou Project Replay Scoreboard

동방프로젝트 리플레이 파일(.rpy)을 업로드하면 스코어와 플레이 정보를 확인할 수 있습니다!
東方Projectのリプレイファイル(.rpy)をアップロードすると、スコアとプレイ情報を確認できます！
Upload your Touhou Project replay files (.rpy) to view scores and play information!

### 사용법

1. 아래 영역에 .rpy 파일을 드래그 앤 드롭하거나 클릭하여 파일을 선택하세요.
2. 여러 파일을 한번에 업로드할 수 있습니다.
3. 테이블 헤더를 클릭하면 정렬할 수 있습니다.
4. 모든 파일 처리는 브라우저에서 이루어지며, 서버로 전송되지 않습니다.
