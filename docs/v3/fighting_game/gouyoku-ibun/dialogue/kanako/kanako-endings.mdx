  <Action>석유 분출은 옛 피의 연못 지옥을 각성시킨 것에 따른, 아마도 자연 현상과도 같은 것이라 여겼다.</Action>
  <Action>피의 연못 지옥을 각성시킨 것은 토테츠인가, 아니면 각성하였기에 토테츠가 들러붙은 것인지는 알 수 없었지만</Action>
  <Action>석유 분출을 멈추기 위해서는 토테츠와 협력할 수밖에 없다고 결론지었다.</Action>
  <Action>즉, 토테츠는 석유가 누출되지 않도록 지하에서 관리한다.</Action>
  <Action>그럼에도 누출된 석유는 화로에서 소각하거나 카나코가 관리한다.</Action>
  <Action>누구에게도 알려지지 않은 암흑 속 지하에서 비밀 계약이 체결된 것이었다.</Action>
