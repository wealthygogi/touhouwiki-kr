# 동방문화첩 (Th095)

**東方文花帖 ～ Shoot the Bullet**

| | |
|---|---|
| 출시일 | 2005년 12월 30일 |
| 개발 | 상하이 앨리스 환악단 (ZUN) |
| 장르 | 사진 촬영 슈팅 |

## 줄거리

:::caution 준비 중
게임 줄거리는 추후 업데이트 예정입니다.
:::

## 특징

- 탄막을 카메라로 흡수해 적에게 반사하는 독특한 사진 촬영 시스템
- 아야만 플레이어블, 공격·회피 불가
- 기존 슈팅 시리즈 보스들이 재등장
