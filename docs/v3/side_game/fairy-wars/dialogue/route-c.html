여기가 바로 외나무다리구나. 일본에서 『百年目(백 년째)』란 『운이 다한 때』, 『끝장인 순간』 등의 의미를 가진다. 즉 『ここで会ったが百年目(여기서 만난 게 백 년째)』는 나를 만났으니 너의 운이 다 했다는 의미의 관용구이다. 우리말에는 같은 의미의 관용구가 없어, 어느 정도 유사한 의미로 해석이 가능하도록 우리말 속담 『원수는 외나무다리에서 만난다』를 변형하여 위와 같이 번역하였다.
