의외의 등장인물과 캐릭터간의 뜻밖에 복잡한 관계성이 보이는 내용이 되었습니다.

약육강식의 세계답게 게임 난이도가 상당하네요. 스토리는 다 클리어해야
완결이기 때문에 꽤 험난한 길이 될 것 같습니다.

엔딩이 가장 깔끔하기 때문에 열심히 해주시기 바랍니다. :-b
