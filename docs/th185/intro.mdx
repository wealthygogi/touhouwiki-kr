# 불릿필리아들의 암시장 (Th185)

**バレットフィリアたちの闇市場 ～ 100th Black Market**

| | |
|---|---|
| 출시일 | 2022년 8월 14일 |
| 개발 | 상하이 앨리스 환악단 (ZUN) |
| 장르 | 탄막 슈팅 |

## 제작자 코멘트

안녕하세요, ZUN입니다.
이번 작품은 100번째 코믹마켓을 기념하여 만들었습니다.
게임 내용도 코믹마켓과 관련해서 정규 시장과는 별도로
뒷거래가 오가는 암시장을 즐기는 이들의 이야기가 되었습니다. (깊은 의미는 없어요)

휴, 뜻밖에 볼륨이 큰 작품이 돼버렸습니다.
스토리만 신경쓴다면 운좋게 좋은 카드를 얻어
비교적 빨리 진상에 도달할 수 있겠지만,
카드를 전부 모으려면 뼈가 부러질 정도입니다. 부러졌어요.

내용은 홍룡동의 속편에 해당하지만 크게 다른 점은 공략의 우연성입니다.
운좋게 벌기도 하고 편할 때도 있고, 불가능할 정도로 어렵기도 합니다.
이번 작품은 단시간의 반복 플레이를 상정해서 일부러 진폭을 크게 했습니다.
단시간이기 때문에 잔기와 봄으로 찍어누르는 것만이
정답이 되지 않도록 최대한 조절했습니다. (그런 공략도 가능하지만요)

공격력을 중시한 즉각적인 효과, 잔기를 중시한 장수 플레이,
벌이 중시로 암시장에 의존하기...... 등등. 다양한 카드 조합으로
공략을 세우는 것은 즐겁지만, 어떤 장비를 착용해도
결국 자신의 실력이 없으면 클리어는 어려울지도 모릅니다.
그러나 절망적인 난이도로 보여도 랜덤 요소가 있으니
운좋으면 쉽게 클리어할 수 있을지도?

음, 이번 작품은 비교적 이상한 매력이 있는 작품이라고 생각합니다.
최종 테스트 플레이도 마치 처음 플레이하는 기분으로 공략할 수 있었습니다.
(클리어까지 힘들었지만
제가 플레이했던 버전보단 쉬워졌으니 안심하세요)
마리사의 인간미 있는 성격도 참 좋으니까
꼭 클리어까지 도전해주셨으면 좋겠습니다.

그리고, 100번째 코믹마켓 축하합니다!
상당히 어려운 시국이 계속되고 있지만, 코로나 사태로 재인식하게 됐습니다.

인간은 생생하게 살아있다는 것을.
물물교환, 즉 물건 매매의 원점을 모르고선
물건을 만들 수 없다는 것을.

그러니 코믹마켓에 참여하는 모든 분에게 행복 가득하길!

ZUN (여기까지 달려왔구만. 운동부족이 극에 달하지만)
